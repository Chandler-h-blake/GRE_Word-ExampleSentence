# README.md

## GRE Vocabulary Notes

这是一个个人 GRE 词汇复习笔记仓库，用来整理备考过程中遇到的重点词汇，并为每个单词补充自行编写的英文例句。

## 内容

- 覆盖范围：Word List 01 - Word List 46
- 整理形式：每个 Word List 单独保存为一个 Markdown 文件
- 文件内容：单词 + 自行编写的英文例句

## 文件结构

计划按下面的方式整理：

```text
WordList1.md
WordList2.md
WordList3.md
...
WordList46.md
```

每个词表文件中，单词和例句采用双列排版，便于快速浏览：

```html
<table>
<tbody>
<tr><td>example</td><td>This is an example sentence.</td></tr>
</tbody>
</table>
```

## 整理原则

- 保留自己的复习顺序，方便按计划推进
- 为每个单词补充一个简洁、自然的英文例句
- 尽量使用适合 GRE 语境的表达
- 发现明显拼写错误时及时修正
- 不收录原书释义、原书例句、助记或讲解内容
- 保持页面清爽，重点放在单词和语境复习上

## 使用方式

可以直接在 GitHub 或本地 Markdown 编辑器中打开对应的 `WordList*.md` 文件复习。

建议复习方式：

1. 先看单词，尝试回忆含义和常见用法。
2. 再阅读例句，确认单词在语境中的使用方式。
3. 对不熟悉的单词单独摘出，进行二次复习。
